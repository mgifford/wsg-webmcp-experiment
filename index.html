Experimental browser interface for the Web Sustainability Guidelines JSON data.

What WebMCP enables

This page works as a normal JavaScript demo in modern browsers. When WebMCP is available, it also exposes selected WSG functions as structured tools that compatible AI agents can discover and call.

The difference is simple: without WebMCP, humans use buttons and forms. With WebMCP, AI agents can call tools such as wsg.search, wsg.get_guideline, or wsg.suggest_audit_questions directly from the page.

Registered WebMCP tools

  • wsg.search: search WSG guidance by keyword, category, or tag.
  • wsg.get_guideline: get a full WSG guideline by ID or exact title.
  • wsg.get_criterion: get a WSG success criterion by generated ID or exact title.
  • wsg.list_by_tag: list guidance connected to a WSG tag.
  • wsg.find_resources: find supporting WSG resources.
  • wsg.suggest_audit_questions: generate draft audit questions.
  • wsg.suggest_procurement_requirements: generate draft procurement requirements.
  • wsg.generate_conformance_claim_draft: generate a draft conformance claim.
  • wsg.stats: show WSG data statistics.
